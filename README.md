# MERN Blog Application

This is a full-stack blog application built using the **MERN stack** (MongoDB, Express, React, Node.js). It allows users to create, read, update, and delete blog posts, as well as authenticate, like, and comment on blogs.

## Features

- **User Authentication**: Users can register, log in, and manage their sessions.
- **Blog Operations**: Users can create, read, update, and delete blogs.
- **Like/Comment System**: Users can like and comment on blogs.
- **Admin Role**: Admin users can delete any blog and manage other users' posts.

## Technologies Used

- **Frontend**: React.js, Axios
- **Backend**: Node.js, Express.js
- **Database**: MongoDB, Mongoose
- **Authentication**: JWT (JSON Web Token)
- **Styling**: CSS
